Interactive 3D Gallery (WebGL)

This interactive 3D gallery serves as a centralized showcase of the key projects I have developed throughout my career. Rather than presenting a conventional portfolio of images or documents, it provides an immersive environment where visitors can explore each project within a real-time 3D space.

Each exhibition represents a different project and can be inspected in detail to examine its architecture, development process, implementation, and the technologies involved. Selected exhibits also function as interactive portals, granting access to fully explorable environments that demonstrate the projects in operation and provide a hands-on understanding of their functionality.

The gallery was developed in Unity and deployed for the web using WebGL, making it directly accessible through a standard browser without requiring additional software. Beyond showcasing software engineering expertise, the project demonstrates proficiency in 3D modeling, environment design, user interface development, lighting, rendering optimization, and user experience design.

This project represents the convergence of software engineering and interactive digital design, creating a continuously evolving platform that expands over time with new projects, technical experiments, and innovative ideas.

Controls:

  • W A S D to Move
  • Mouse to Look Around
  • E to Inspect Exhibit
  • Hold LMB to Rotate Exhibit
  • Q Return
